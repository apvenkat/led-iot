Login into your Raspberry pi and open the text editor and create a file led.py

sudo nano led.py    //create file

copy and paste the given code or clone the repository in any of your raspberry pi folder

Connect your raspberry pi in GPIO4 pin

execute using the command

sudo python led.py     //execute

You can see your led turn on and off, Now You can change lot of logics and observe the behaviour of your LED..!!!!!
